# InnerGlowYoga - Static HTML Website

This folder contains the complete conversion of the InnerGlowYoga React website into standalone HTML, CSS, and JavaScript files.

## Files Included

### HTML Pages (6 total)
1. **index.html** - Home page with hero section, features, benefits, and testimonials
2. **classes.html** - Yoga class descriptions with detailed information and pricing
3. **about.html** - About page with instructor profiles and philosophy
4. **contact.html** - Contact form with location map and FAQ
5. **terms.html** - Terms & Conditions legal document
6. **privacy.html** - Privacy Policy legal document

### CSS & JavaScript
- **style.css** - Global stylesheet with emerald green and beige color scheme
- **script.js** - Interactive features (mobile menu, forms, navigation)

## How to Use

1. Open any `.html` file in a web browser
2. All files are standalone - no build process needed
3. Navigate between pages using the header menu
4. All links and forms are functional
